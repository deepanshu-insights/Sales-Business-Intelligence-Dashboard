# Sales-Business-Intelligence-Dashboard

* Engineered a complete BI pipeline by ingesting and reshaping raw CSV data spanning sales, returns, products, customers, and territories via Power Query.
* Modeled relationships across tables and authored DAX measures to monitor core KPIs — revenue, profit, return rates, and overall sales — for leadership reporting.
* Built interactive dashboards enabling side-by-side regional comparisons and deep-dives into product performance to guide strategic planning.
* Uncovered high-value customer profiles through demographic and geographic visual analysis, supporting more precise targeting.
